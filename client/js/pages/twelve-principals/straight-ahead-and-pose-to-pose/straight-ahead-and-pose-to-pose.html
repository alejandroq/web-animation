Straight Ahead & Pose to Pose

  • Doesn't apply a whole lot to web.
  • Pose to Pose is drawing the key frames, and then filling in between.
  • Straight ahead is drawing frame after frame.

CSS animation is basically pose to pose.

@keyframes fontbulger {
  0% {
    font-size: 10px;
  }
  30% {
    font-size: 15px;
  }
  100% {
    font-size: 12px;
  }
}
